# Features
- Simple configuration
- Logs (coming soon)
- TS built-in (coming soon)

# Installing
```batchfile
git clone https://github.com/J-P-S-O/Http-Server
cd HTTP-SERVER
npm run get-dependencies
tsc
node main.js
```
